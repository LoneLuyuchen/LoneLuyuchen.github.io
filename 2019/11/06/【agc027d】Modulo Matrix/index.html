【agc027d】Modulo Matrix

2019-11-06
构造

题目大意：

现在需要你构造一个$n\times n$的矩阵，满足其中任意两个数不相等，

并且对于相邻两个数$x,y$，$max(x,y)mod\ min(x,y)$为一个常数。

<!--more-->

解题思路：

我们容易想到，将矩阵黑白染色，

不妨设白色部分比周围四个格子中的数都要大。

黑色部分随便填，白色部分填上周围四个数的$LCM$

但是这样会挂掉因为答案会比$10^{15}$大。

所以我们考虑怎么做才能让$LCM$变小。

对于每个黑色的格子，我们考虑斜着考虑，对于第$i$个左斜行，第$j$个右斜行，它的答案为所对应的两个素数的乘积。

这样就可以大大减小$LCM$.
